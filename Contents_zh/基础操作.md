
# 基础操作

Dynamaker网址链接: https://dynamaker.tunergames.com/

###### 欢迎来到玩家自制的Dynamaker使用教程！
Dynamaker是由omegaPi开发的，能让你自己制作Dyanamix谱面的网页制谱器

## 第一步：让Dynamaker适配你的屏幕大小 

推荐使用 *Google Chrome* 访问Dynamaker  
按下F以进入全屏模式。 （也可以按F11，这样能让网页元素对齐）


**如果网页并不适配你的屏幕大小:**

设置 **网页缩放** 为100%   
	- （使用Ctrl+鼠标滚轮来调整网页大小。如果这依旧不适配，请尝试其他缩放大小如80%，直到界面差不多对齐）


## 第二步：选择你的音乐文件（和谱面文件）

如果你在创建一个新的谱面，只需选取音乐文件后点击 **New Map** 即可。  
	- 点击 *Audio/Video Browse* 来上传音乐文件。当你鼠标移动到选项上时，文字理应来说会变成绿色，此时点击即可加载音乐文件。如果选项还是蓝色无反应，可能因为你的网页没有对齐。尝试在选项附近移动来选中选项或者调整网页缩放，之后的选项也是同理。 

如果你要编辑已经存在的谱面，你需要同时上传音乐文件和谱面文件。


## 第三步：设置谱面配置

通过第三方BPM测试器来确定音乐文件的BPM大小。如果你事先知道音乐的延迟，你可以输入相应的数字到offset，如果不知道就保留为0。你可以在之后的编辑中使用 **"P"** 和 **"O"** 来调整音乐延迟。

  - Beat Per Minute 代表音乐的BPM大小。
  - Bars Per Minute 代表BPM除以歌曲的拍号（每小节拍数）大小，如4/4拍则在Bars Per Minute中输入BPM除以4后的大小。绝大部分情况下Bars Per Minute中都是填写BPM除以4后的数字。

##### 左右侧设置：

  - MIXER: 侧边为滑板，只有Slide Note（即红Note）
  - PAD: 侧边为普通侧边，只有普通Note（即蓝Note）和Hold（即长条）
  - MULTI: 可以同时包含以上三种Note的侧面 （这种模式在Dynamix中是不存在的）


## 第四步：创建谱面

点击 Start 以创建新谱面。
使用 **"P"** 和 **"O"** 来调整音乐延迟（Offset），其中 **O** 能让Note晚到判定线，**P** 能让Note早到判定线。你可以在比较低的音乐播放速度下调整音乐延迟来让谱面对齐音乐，如何调整音乐速度请在快捷键及参数一栏中查看。

## 如何保存谱面
1. 右键
2. 点击 Save as XML 来保存谱面
3. 之后如果你想再次打开Dynamaker编辑这个谱面，只需要导入你所保存的谱面所对应的XML文件和音乐即可

你也可以选择 Save as .DY 来在一个文件里同时保存谱面和音乐，这样在第二次打开时你只需要上传。DY文件即可，不再需要单独上传音乐文件。但是这样得到的谱面文件大小会变得比较大，如果你频繁保存。DY文件可能会占据很多存储空间。  

.DY文件也能上传至你的移动端Dynamaker网页中，然后在网页中游玩谱面。你可以通过这种方式来不用上传谱面至Dynamite就能测试你的谱面。但是由于在部分设备上网页尺寸可能不适配屏幕大小，这种方法只对部分设备有效。
