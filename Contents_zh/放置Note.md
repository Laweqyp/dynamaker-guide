
# 来尝试放一些Note吧!

按下 **1** 以选择 *Tap Notes* 模式，之后 **鼠标左键** 来在你当前光标处放置一个Tap Note。 

在放置Note时，长按 **鼠标左键** 然后左右拖动可以调整Note长度。 

如果你想放置红条，只需要按下 **2** 选择 *Slide Notes* 模式，然后的操作和放置Tap Notes的操作一样。   

按下 **3** 选择 *Hold Notes* 模式来放置长条，  
单击 **鼠标左键** ，你会在当前光标处放置长条的起点。之后再移动鼠标选择另一处你想让长条结束的地方，并再次按下 **鼠标左键** 来放置长条末尾。

按下 **向上方向键** 来切换 **操作面**。不同的操作面决定了你能放置Note的区域（正面，左侧，右侧）。操作面切换顺序为顺时针方向，从正面到左侧再到右侧，最后回到正面，以此循环。  

按下 **4** 进入 **EDIT Mode** 来移动或者删除你已经放置的Note。（具体操作方法请参考[快捷键及操作](Controls/快捷键及操作.md)）

如果你想删除Note，你可以按下 **4** 进入 **Edit Mode**，然后 **鼠标左键** 拖动选择区域并选中你想删除的Note，之后按下 **Del** 或者 **右键** 呼出菜单来删除Note。  
注意删除操作只会删除当前所在操作面的Note，例如当你在编辑正面时，框选删除只会删除正面选择的Note，不会删除侧面的Note。当你在侧边编辑时也只会删除侧边的Note，不会删除另外两边的Note。

请务必查看[快捷键及操作](Controls/快捷键及操作.md)这一章节来了解如何调整歌曲播放进度以及更多的操作。

## 光标周围的参数

下面这个例子说明了Note周围的一些参数：

        *第N拍 + x/y*
                <=============>
        *水平位置*             *Note大小*

**这些参数一开始很容易搞混，并不需要一定掌握：**

**第N拍 + x/y**: 第N拍代表了你当前放Note的地方是第N个小节拍。之后的 x/y 涉及一些乐理，  
y 代表了一个小节被y等分（可以通过 **C/V** 切换，推荐使用4分或8分，使用12分来放置一些4分或8分所放不了的音符，如24分）  
x 代表了当前位置为当前小节的第x个y等分点

y值越大，意味着每一分割之间所间隔的时间更短，  

对于一首4/4拍的歌，你可以设置y=4然后在1/4、 2/4、 3/4、 4/4拍的位置放置note，或者你可以设置y=16并在4/16, 8/16, 12/16, 16/16处放置note得到和前者一样的效果，但是y=16代表你可以相比y=4在同一小节中放更多的note  

可以理解为x/y代表了你放置note的时间戳大小，但是时间戳的单位为拍数。


**水平位置** ：Note所在位置的x坐标值



<h2 id="multi-colour">在放置Note时出现在Note中间的色块：</h2>
**白色色块**:  
出现在 EDIT Mode（当你按下 **4** 时）里。当你光标在note附件徘徊时就会出现，你可以双击选中note。  

![Pic Showing Blue, Red and Purple Square on Note](Images/Blue%20Red%20Purple%20Squares.jpg?raw=true "Blue, Red and Purple Square")

**蓝色色块**:  
当你放置note调整note大小时，或者是你选择了一个note时，和这个note大小相同的note会一起出现蓝色色块，有蓝色色块的note即代表了这些note有一样的大小，

**红色色块**:
当你移动note时，如果其他note和这个note都出现了红色色块，则说明当前移动的位置的垂直位置和其他有红色色块的note的中心位置是对齐的。在对齐note时候十分好用。

**紫色色块**:
包含了蓝色色块和红色色块的两个意思，出现紫色色块的note都有一样的大小和垂直位置。（紫色色块=蓝色色块+红色色块）


### 功能不清楚的色块：

<h1 id="troubleshooting">故障处理</h1>

如果你在制做谱面时遇到任何BUG导致屏幕变黑或者报错，请按以下步骤来恢复你的谱面，不要浪费你过去几个小时的努力!  
_（比如： 你按 **S** 太多次导致歌曲速度变成0）_

#### 步骤：
1. 按下 "F12"
2. 在弹出的开发者工作栏的顶部选择"Console"
3. 输入 save() ;

然后你可以在弹出的窗口中选择xml谱面文件的保存位置并下载文件。之后只要重新加载Dynamaker，然后重新打开下载的xml文件，就可以稍微松一口气了。
