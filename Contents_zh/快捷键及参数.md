
# 基础控制操作


## 播放参数

键盘：

**A / D / 滚轮** : 前进/后退一秒歌曲 （让谱面上下移动）

**Shift + A / Shift + D** : 前进/后退0.01秒歌曲


**Q / E** : 设置note下落速度（和Dynamix里的下落速度一样）

**W / S** : 设置歌曲播放速度
- 正常播放为 1.0
- 放慢播放速度来更方便的调整谱面延迟和对准note音

_BUG: 不要频繁按S ，这可能会导致播放速度变为0并导致Dynamaker崩溃。_
_如果你遇到了这种情况，查看[故障处理](放置Note.md/#Troubleshooting)来保存你的谱面_

**O / P** : 调整谱面延迟。 按O让note慢点到判定线，P反之亦然

**Enter** : 进入自动谱面播放模式，会显示分数和Combo数（和Dynamix游玩界面一样）.

**R / M** : 从头播放谱面（没有分数和Combo数）

**Space** : 播放/暂停歌曲



## 制谱

### 鼠标:

**左键**: 放置/调整note大小/位置

**右键** : 打开菜单

### 键盘:

**向下/左/右方向键** : 打开/关闭正面/左侧/右侧时间轴 
- 按一下打开50%透明的的时间轴
- 第二次按下打开不透明的时间轴
- 第三次按下关闭时间轴


##### 比较重要但是容易搞混的东西：

**向上方向键** : 切换操作面：

	顺序：正面 --> 左侧 --> 右侧
	
操作面决定了你能放置Note的区域，你可以且只可以在当前操作面放置note。当你进行其他操作时（比如删除note），只会影响当前操作面的note，不会影响别的操作面的note。

**1 / 2 / 3** : 选择note种类

1: Tap（蓝条，单点）

2: Slide（红条，划键）

3: Hold（黄条，长按）


**4** : 进入 **Edit** 模式

<h1 id="edit_mode">EDIT 模式</h1>

在 EDIT Mode 下，当你光标在note附件徘徊时会出现白色色块，你可以双击选中并移动note到别的地方。 

你可以通过左键拖动同时选择多个note，形成的选择区域就能同时包含了这些note。

（只会选中当前操作面的note）

**Del** : 在 **Edit Mode** 下选中note后按下来删除这些选中的note

**H** : 按住不放可以弹出一些帮助文字，这些文字介绍了一些其他的功能。

<h1 id="random_stuff">在note周围的一些细长条：</h1>

**白线** : 能告诉你如果你左键单击放置note，你的note将被放置在哪一列。这可以用来帮助你来对齐你的note。

**蓝/红/黄线** : 能告诉你如果你左键单击放置note，你的note将被放置在哪一时间戳。这可以用来帮助你对音。比如，如果你把note放在右边，左边也会有这样一条线，帮助你比较左右两边note的时间。  

**紫线** : 显示当前操作面的位置所代表的时间在其他面的一样的时间的位置。和 **蓝/红/黄线** 差不多。

- 用这个来判断你当前所放的note与其他在不同面的note是不是在同一时间，多用来摆放双/多押。


**note周围的数字** : 见[光标周围的参数](放置Note.md/#niche).

**出现在note中间的色块** : 见[在放置Note时出现在Note中间的色块](放置Note.md/#multi-colour).
