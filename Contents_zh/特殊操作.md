
# 一些你永远也不会知道的实用操作

## 实用 - 撤销 & 恢复 
- 使用前保存文件，这可能会让DYNAMAKER崩溃

键盘:

**Shift + 向左方向键** : 撤销 （删除最近一次放置的note/将note的移动撤销）

**Shift + 向右方向键** : 恢复 （和撤销相反）



# 直到你试过了才清楚的功能

键盘:

**C / V** : 改变每一小节切分的note个数，按 **C** 减少，按 **P** 增加

**Shift + C / V** : 和 **C / V**功能类似。按 **Shift + C** 微量减少，按 **Shift + P** 微量增加

**长按 Z**: 显示note所对应的时间戳（单位为小节而非秒）xx

**长按 X** : 更精细的note摆放位置操作  


**Shift + R** : 设置歌曲速度为0.1，音符速度为1.0，并切换至1/32分（可能方便用来校对offset？）.


**L** : “画笔”功能。方便绘制同一条线上的多个note（密度基于每小节的分割数）
